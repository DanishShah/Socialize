# Socialize
A Social Networking Website made as a college project. There are lot of functions to be implemented. I might work on this project in future.

<a href="www.socialize.16mb.com">Check out the website</a>

# Functions
* Login/Sign up.
* Create Post.
* Like Post.
* Comment Post.
* Follow/Unfollow.
* View Profile.
